# 灵动陪伴 - AI情感陪伴应用

> 一个温暖的AI伙伴，陪伴你记录生活、分享心情、获得情感支持

基于React Native和DeepSeek API开发的AI情感陪伴应用，提供智能聊天、日记记录、社交分享和心情分析功能。

## ✨ 功能特色

### 🎯 核心功能
- **🏠 智能球体交互**: 按住球体进行语音对话，AI实时回复，20种心情色彩
- **📱 社交帖子系统**: AI生成互动帖子，智能推荐，积极反馈机制
- **📔 智能日记系统**: 自动分类标签，AI日记生成，心情分析
- **📊 心情统计分析**: 可视化情感趋势，AI洞察建议

### 🎨 设计亮点
- **流畅动画**: 60fps丝滑动画，自然过渡效果
- **手势导航**: 上滑帖子，右滑日记，继续右滑心情分析
- **动态主题**: 首页动态背景，其他页面纯白简洁
- **半透明效果**: 高斯模糊，现代化毛玻璃效果

### 🤖 AI特性
- **完整记忆**: AI记住所有对话历史和用户偏好
- **情感识别**: 实时分析20种情感状态
- **个性化适应**: 根据用户习惯动态调整回复风格
- **积极陪伴**: 所有AI互动都是温暖正面的

### 🔒 隐私保护
- **完全本地存储**: 所有数据保存在设备本地
- **无数据上传**: 仅与DeepSeek API通信
- **隐私安全**: 不收集任何个人信息

## 🚀 快速开始

### 📱 用户使用 (推荐)
1. **下载APK**: 获取编译好的Android APK文件
2. **安装应用**: 在Android设备上安装
3. **设置API**: 首次打开设置DeepSeek API密钥
4. **开始陪伴**: 享受AI情感陪伴体验

### 🛠 开发环境
- Node.js 16+
- React Native CLI (不使用Expo)
- Android Studio
- DeepSeek API密钥

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <项目地址>
   cd ai-companion-app
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **Android开发**
   ```bash
   npx react-native run-android
   ```

### API密钥配置

访问 [platform.deepseek.com](https://platform.deepseek.com/api_keys) 获取免费API密钥：
1. 注册DeepSeek账号
2. 创建API密钥
3. 在应用中输入密钥

## 📁 项目结构

```
src/
├── components/          # 可复用组件
│   ├── SoulBall.js     # 智能球体 (20种心情色彩)
│   └── PostCard.js     # 帖子卡片组件
├── screens/            # 页面组件
│   ├── ApiKeySetup.js  # API密钥设置 (含跳转链接)
│   ├── MainApp.js      # 主应用 (丝滑手势导航)
│   ├── HomeScreen.js   # 首页 (智能球体交互)
│   ├── PostsScreen.js  # 帖子 (搜索/通知/收藏)
│   ├── DiaryScreen.js  # 日记 (分类/AI日记)
│   └── MoodScreen.js   # 心情统计分析
├── services/           # 服务层
│   ├── AIService.js    # AI服务 (DeepSeek集成)
│   └── StorageService.js # 本地存储 (完整数据管理)
└── theme/              # 主题配置
    └── theme.js        # 颜色和样式
```

## 🔧 技术栈

- **框架**: React Native (纯原生，不使用Expo)
- **UI库**: React Native Paper + React Native Reanimated
- **存储**: AsyncStorage (完全本地化)
- **AI服务**: DeepSeek API (非思考模式)
- **语音**: React Native Voice
- **动画**: React Native Reanimated + Gesture Handler
- **图标**: React Native Vector Icons

## 🎯 核心功能详解

### 🏠 首页 - 智能球体
**⚠️ 重要：请不要修改首页的核心布局设计**

- **半透明球体**: 高斯模糊效果，支持交互状态变化
- **20种心情色彩**: 根据AI情感实时变化颜色
- **语音交互**: 按住说话，松开获得回复
- **流式文字**: 逐字显示，标点停顿，点击加速
- **动态背景**: 根据心情变化的渐变背景

### 📱 帖子系统
- **智能生成**: 每次进入生成10个AI帖子
- **智能推荐**: 基于浏览记录的个性化推荐
- **搜索功能**: 搜索帖子内容和作者
- **通知系统**: 点赞评论通知
- **收藏功能**: 收藏喜欢的帖子和评论

### 📔 日记系统
- **自动分类**: AI根据内容自动打标签
- **自动命名**: 未命名日记AI生成标题
- **智能摘要**: 长日记自动生成简介
- **AI日记**: AI每天写自己的日记
- **瀑布流**: 卡片式展示，轻微错位美化

### 📊 心情分析
- **多源数据**: 聊天、日记、帖子情感数据
- **可视化图表**: 心情分布和趋势分析
- **AI洞察**: 情感状态分析和建议
- **时间筛选**: 查看不同时间段的心情变化

## 🔄 手势导航

```
        📱 帖子页面

    📔 日记 ← 🏠 首页 (核心布局请勿修改)
```

## 💾 数据管理

### 本地存储结构
- `conversation_history`: 对话历史
- `posts`: 帖子数据
- `diary_entries`: 用户日记
- `ai_diaries`: AI日记
- `mood_entries`: 心情记录
- `notifications`: 通知数据
- `favorites`: 收藏内容
- `viewed_posts`: 浏览记录

### 数据安全
- 完全本地存储，不上传云端
- 卸载应用会清除所有数据
- 支持数据导出功能

## 🎨 UI/UX设计原则

### 视觉风格
- **首页**: 动态渐变背景 (#0A0A0F → #1A1A2E → #16213E)
- **其他页面**: 纯白背景 (#FFFFFF)
- **主色调**: #6C63FF (紫色)
- **强调色**: #FF6B9D (粉色)
- **圆角**: 统一20px圆角设计

### 动画要求
- **60fps流畅**: 所有动画必须丝滑
- **自然过渡**: 页面切换使用弹性动画
- **即时反馈**: 交互操作立即响应
- **细节动画**: 微交互增强用户体验

## 🔧 开发指南

### 添加新功能
1. 在 `services/` 中添加数据处理逻辑
2. 在 `screens/` 中创建页面组件
3. 更新 `MainApp.js` 导航配置
4. 添加流畅的动画效果

### 自定义AI回复
修改 `AIService.js` 中的系统提示词来调整AI性格。

### 性能优化
- 使用 `React.memo` 优化组件渲染
- 合理使用 `useCallback` 和 `useMemo`
- 动画使用原生驱动 `useNativeDriver: true`

## 📝 开发注意事项

- **首页布局**: 请不要修改首页的核心布局和交互设计
- **Android优先**: 专注Android端开发，不考虑iOS适配
- **本地存储**: 所有数据必须存储在本地
- **AI非思考**: 统一使用DeepSeek非思考模式
- **流畅动画**: 确保所有动画60fps流畅运行
