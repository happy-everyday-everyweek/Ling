# 🚀 灵动陪伴 - 快速部署指南

## 📱 通过GitHub Actions自动构建APK

### 方法一：自动触发构建
每次推送代码到main分支时，GitHub Actions会自动构建APK：

1. **推送代码**：任何对main分支的推送都会触发构建
2. **等待构建**：构建过程大约需要5-10分钟
3. **下载APK**：构建完成后可以从以下位置下载：
   - **Releases页面**：https://github.com/happy-everyday-everyweek/Ling/releases
   - **Actions页面**：https://github.com/happy-everyday-everyweek/Ling/actions

### 方法二：手动触发构建
如果需要立即构建APK：

1. 访问：https://github.com/happy-everyday-everyweek/Ling/actions
2. 点击"构建灵动陪伴APK"工作流
3. 点击"Run workflow"按钮
4. 选择分支（通常是main）
5. 点击绿色的"Run workflow"按钮

### 📥 下载构建好的APK

#### 从Releases下载（推荐）
1. 访问：https://github.com/happy-everyday-everyweek/Ling/releases
2. 找到最新版本
3. 下载`灵动陪伴-v*.apk`文件

#### 从Actions下载
1. 访问：https://github.com/happy-everyday-everyweek/Ling/actions
2. 点击最新的成功构建
3. 在"Artifacts"部分下载"灵动陪伴-APK"

## 📲 安装APK到Android设备

### 准备工作
1. **启用未知来源**：
   - 设置 → 安全 → 未知来源（允许安装来自未知来源的应用）
   - 或者：设置 → 应用和通知 → 特殊应用访问 → 安装未知应用

### 安装步骤
1. 将APK文件传输到Android设备
2. 使用文件管理器找到APK文件
3. 点击APK文件
4. 按照提示完成安装
5. 安装完成后，在应用列表中找到"灵动陪伴"

## ⚙️ 首次使用配置

### 1. 获取DeepSeek API密钥
1. 访问：https://platform.deepseek.com/
2. 注册账号并登录
3. 进入API密钥管理页面
4. 创建新的API密钥
5. 复制密钥（格式类似：sk-xxxxxxxxxxxxxxxx）

### 2. 配置应用
1. 打开"灵动陪伴"应用
2. 首次启动会显示"给我注入灵魂吧"页面
3. 在输入框中粘贴你的DeepSeek API密钥
4. 点击确认完成配置

## 🔧 构建状态监控

### 查看构建状态
- **构建徽章**：在README中可以看到构建状态
- **Actions页面**：https://github.com/happy-everyday-everyweek/Ling/actions
- **通知**：GitHub会在构建失败时发送邮件通知

### 构建失败排查
如果构建失败，可以：
1. 查看Actions页面的错误日志
2. 检查代码是否有语法错误
3. 确认所有依赖都已正确配置
4. 重新触发构建

## 📋 版本管理

### 版本号规则
- 格式：`v1.0.{构建号}`
- 每次构建会自动递增构建号
- 例如：v1.0.1, v1.0.2, v1.0.3...

### 发布说明
每个版本的Release都包含：
- 📱 APK下载链接
- 🆕 功能特性说明
- 📝 安装指南
- 🔄 更新内容

## 🚨 注意事项

1. **网络要求**：构建过程需要稳定的网络连接
2. **构建时间**：通常需要5-10分钟，请耐心等待
3. **存储空间**：APK文件大小约20-50MB
4. **兼容性**：支持Android 5.0（API 21）及以上版本
5. **权限**：应用需要网络、存储、麦克风权限

## 🆘 常见问题

### Q: 构建失败怎么办？
A: 查看Actions页面的详细日志，通常是依赖安装或网络问题

### Q: APK安装失败？
A: 确保已启用"未知来源"安装，并检查设备存储空间

### Q: 应用闪退？
A: 确保API密钥正确配置，检查网络连接

### Q: 如何更新应用？
A: 下载新版本APK，直接安装即可覆盖旧版本
